# Gameup

# Opis aplikacije
Gameup je veb aplikacija koja nudi pregled video igrica, kao i gledanje njihovih trailer-a.

# Korisnici
- Neulogovani korisnik: može da pretražuje, filtrira, vidi detalje igrice
- Admin: može sve što i neulogovani, uz dodavanje, ažuriranje i brisanje video igrica

# Tehnologije
- Angular - frontend
- Rust - backend
- PostgreSQL - baza podataka
